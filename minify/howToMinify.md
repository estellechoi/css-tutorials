### Install and Use 'clean-css-cli' in Mac Terminal.
    `sudo npm install -g clean-css`
    
    `sudo npm install -g clean-css-cli`
    
    `cleancss -o original_file.min.css original_file.css`
    
    `cat original_file.min.css`
